## Source and evidence

This run tests whether the square-root support family is still underweighting reliable PPM-A disagreement.

JS disagreement proxies non-redundant signal; support calibrates reliability. Lecture 13's recurrence view motivates trusting stronger repeated-context evidence while not discarding moderately low-support recurrence.

## Course-material connection
The predictor remains strictly sequential: train counts and past public symbols only, with updates after scoring. This is deterministic expert-gate arithmetic and preserves official evaluator semantics.

## Hypothesis
If Run 057 still underweights square-root-supported PPM-A disagreement, boost `0.68` will improve public log-loss. If Run 057 is near saturation, this will over-amplify mismatched contexts and regress.

## Risks
Gains are tiny and the search is close to public-practice calibration. Keep only on official primary improvement and confirm immediately.
