## Source and evidence
Runs 044, 046, and 048 formed a monotone improvement sequence; Runs 045, 047, and 049 confirmed each kept model exactly. This run tests whether the disagreement boost still underweights PPM-A's incremental signal or whether we have reached the overconfidence/saturation boundary.

Course evidence: Lecture 4's conditional mutual information section says a new signal matters only through uncertainty reduction left after the baseline context is known, and the chain rule decomposes total information into baseline plus extra information. Here JS disagreement is an empirical proxy for PPM-A information not already present in the CTW/VOMM/PPM-C base. Lecture 4's Jensen discussion motivates using mixture/disagreement quantities, but the official log-loss decides whether the proxy is calibrated.

## Course-material connection
The predictor remains strictly sequential: all probabilities use train data and the already observed public prefix only, and updates occur after scoring. The run changes only deterministic expert-gate arithmetic.

## Hypothesis
If PPM-A disagreement continues to correspond to useful conditional information, boost `0.32` may improve over Run 048. If Run 048 is near the safe limit, this will amplify mismatched rare-context overconfidence and regress.

## Risks
This is explicitly a saturation test, so overfitting risk is higher than the earlier sign tests. Keep only on official primary improvement and confirm immediately.
