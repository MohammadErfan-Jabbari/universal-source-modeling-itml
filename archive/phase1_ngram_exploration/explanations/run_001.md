## Course-material connection
The score is empirical cross-entropy in bits/symbol, equivalent to the ideal average code length under the predictor probabilities. The n-gram model is a finite-context source model; its redundancy comes from mismatch between its conditional probabilities and the true source conditionals.

## Hypothesis
The official baseline should reproduce the known score and serve as a stable reference. Future changes must beat this while keeping valid probabilities, sequential behavior, and acceptable runtime.

## Risks
No implementation risk, but there is a methodological risk: optimizing only against the public practice set can overfit. This baseline should be used as a reference, not as proof of live-day performance.
