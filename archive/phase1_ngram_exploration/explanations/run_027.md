# Run 027 — equal mixture of fixed-C and adaptive-C half-count predictors

## Proposed change
Add `submissions/ngram_interpolated_n4_laplace05_model_mix.py`. It computes two valid half-count n=4 interpolated distributions from the same past counts:

1. fixed-confidence model from Run 018: `C=44`;
2. adaptive-diversity model from Run 023: `C_eff = 4 * T(context)`.

The final prediction is their 50/50 probability mixture before taking log2.

## Source and evidence
- source: experiment observation — Run 018 fixed `C=44`, `Laplace=0.5` scored `2.9773145652`; Run 023 adaptive `C_eff=4T` improved slightly to `2.9771815925`.
- source: experiment observation — Runs 024-025 bracketed the adaptive scale and found scale 4 best; Run 026 showed removing local half-count smoothing is harmful.
- source: course material — `lectures/15_17032026_UniversalCompressionIII_MinimaxRedundancy_study.md` says robust universal coding distributions are mixtures over candidate source laws; `lectures/5_04022026_Inequalities_study.md` states cross-entropy excess is mismatch cost.

## Course-material connection
This is an explicit model average over two sequential coding distributions. The fixed-C model and adaptive-C model represent two plausible assumptions about when a context is trustworthy. Mixing them is a universal-coding hedge: if either model is locally mismatched, the other can keep probability mass on the realized symbol. The mixture still uses only the training data and past test symbols.

## Hypothesis
The fixed-C and adaptive-C predictors may make complementary calibration errors. A 50/50 mixture can reduce worst-case mismatch across contexts and may beat the adaptive model alone. If Run 023 strictly dominates the fixed-C model in most contexts, the mixture will dilute it and worsen slightly.

## Risks
This adds one more model-average layer and a few vector operations per token, but runtime should remain far below the limit. The public-practice gain, if any, may be small; after this, validation/repeat confirmation should take priority over more mixture-weight tuning.
