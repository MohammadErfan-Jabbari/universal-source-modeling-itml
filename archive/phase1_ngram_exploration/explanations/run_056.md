## Course-material connection
The mixture weight controls how much coding probability is assigned to a test-realization-specific low-order source versus the train-heavy n=5 model. If the all-past n=3 online source better matches the evaluated realization, increasing its weight reduces mismatch. If it is over-weighted, the code gives up too much long-context/train evidence and redundancy increases.
