## Course-material connection
The mixture weight controls a universal-coding hedge between a train-heavy longer-context model and a test-realization-specific low-order online model. Because cross-entropy excess is mismatch cost, the optimal weight is where online adaptation reduces mismatch without discarding too much reliable long-context evidence. The asymmetric bracket suggests `0.15` may be slightly high.

## Hypothesis
If the true all-past `n=3` online mixture optimum is slightly below `0.15`, then weight `0.14` will beat `2.9719738362`. If the run 057 gain was already at the optimum or within public-practice noise, `0.14` will be unchanged or worse.

## Risks
This is narrow public-practice tuning with a likely tiny effect, so any improvement should be validated or repeated later. Runtime should match Run 057. The predictor remains strictly sequential, uses no external calls, and does not modify evaluator semantics.
