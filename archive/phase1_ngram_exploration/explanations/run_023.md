# Run 023 — n=4 half-count interpolation with unique-symbol adaptive C

## Proposed change
Add `submissions/ngram_interpolated_n4_laplace05_unique_c4.py`. It keeps `n=4`, half-count local smoothing, and online adaptation, but replaces the fixed interpolation constant with a context-dependent value:

```text
C_eff(context) = 4 * T(context)
lambda = total / (total + C_eff)
```

where `T(context)` is the number of distinct next symbols observed after that context. Contexts with concentrated next-symbol support are trusted sooner; contexts with many distinct next symbols keep more lower-order mixture mass.

## Source and evidence
- source: experiment observation — Run 018 remains best with fixed `C=44`, `Laplace=0.5`; Runs 019-022 bracketed smoothing/C and showed narrow tuning is now saturated.
- source: ideas backlog — `autoresearch.ideas.md` proposes PPM-style predictors with escape probability based on the number of unique seen symbols and total counts.
- source: agent prior knowledge — Witten-Bell/PPM-style escape heuristics use the number of distinct continuations as an uncertainty signal; more distinct outcomes imply more escape/backoff mass.
- source: course material — `lectures/13_10032026_UniversalCompressionI_LempelZiv_study.md` says repeated past patterns are evidence about probabilities; `lectures/15_17032026_UniversalCompressionIII_MinimaxRedundancy_study.md` motivates robust mixture coding distributions; `lectures/5_04022026_Inequalities_study.md` ties excess cross-entropy to mismatch.

## Course-material connection
The fixed-C model uses only total context count to decide how much to trust a higher-order model. A universal/PPM-style view also asks how diverse the observed continuations are: a context seen 50 times with one continuation is more predictive than a context seen 50 times with all 16 continuations. This run changes the mixture weight using only past counts, so it remains a valid sequential probability assignment.

## Hypothesis
Adaptive `C_eff=4T` may improve calibration by giving more weight to higher-order contexts that have low continuation diversity, while backing off more for ambiguous contexts. If diversity is a noisy proxy or the scale is wrong, the fixed `C=44` half-count model will remain better.

## Risks
The scale `4` is heuristic and may overtrust low-diversity contexts. This is qualitatively different from fixed-C tuning but still uses public practice as the primary metric, so any small gain should later be validated on train-derived splits. Probabilities remain positive because local half-count distributions and lower-order mixtures are normalized.
