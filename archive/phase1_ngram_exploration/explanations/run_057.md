## Course-material connection
This run tests model averaging: how much probability mass should be allocated to a test-realization-specific low-order model versus the train-heavy longer-context model. If the all-past n=3 model reduces mismatch on the current source realization, a larger mixture weight lowers empirical codelength; if over-weighted, it discards useful long-context evidence and raises redundancy.

## Hypothesis
If weight `0.12` still underweights the stable all-past n=3 component, weight `0.15` will improve. If `0.12` is near the optimal hedge, `0.15` will dilute the n=5 main model and worsen.
